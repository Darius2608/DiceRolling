# DiceRolling
I created a program which simulates rolling multiple dice using ASCII art and then adding up the total number shown on each dice. 
The user is prompted to enter the number of dice to roll and then program will display an ASCII representation for each face. 
A running total is calculated and a sum of the dice is displayed to the user.
I have used input validation to make sure user enters valid number of dice, a static method so that code is more organised and each logic has its own block of code, and switch expressions for decision making.
